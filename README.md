# REST API Count Magic Number

[![Build Status]()

This is a basic model of an api created with Java 11 and Spring Boot, guided by unit and integration tests


## Install

    mvn clean install

## Run the app

    mvn spring-boot:run

## Run the tests

    mvn test

## Run Docker Compose

    docker-compose up

## Route
    POST localhost:8080/api/magicnumber
## Documentation
    localhost:8080/swagger-ui.html
